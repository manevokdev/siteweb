# Structure des fichiers à ajouter

## Créez ces dossiers dans le répertoire `public/` :

### Pour les publications
- `public/publications/` (publications françaises)
- `public/publications/en/` (publications anglaises)
- `public/publications/thumbs/` (miniatures des publications françaises)
- `public/publications/thumbs/en/` (miniatures des publications anglaises)

### Pour les formations
- `public/formations/` (programmes de formation)

### Pour les webinars
- Placez les images de webinars directement dans `public/`

## Exemple de contenu à ajouter :

1. **Publications françaises** (à placer dans `public/publications/`) :
   - pic-pdp-exo.pdf
   - securisation.pdf
   - veille.pdf
   - negociation.pdf
   - cdc.pdf
   - continuite.pdf
   - quizz.pdf

2. **Publications anglaises** (à placer dans `public/publications/en/`) :
   - sop-mps-exo.pdf
   - supplies.pdf
   - intelligence.pdf
   - negociation.pdf
   - specifications.pdf
   - continuity.pdf

3. **Images de miniatures** (optionnel, pour améliorer l'apparence) :
   - `public/publications/thumbs/pic-pdp.jpg`
   - `public/publications/thumbs/securisation.jpg`
   - etc.

4. **Programmes de formation** (à placer dans `public/formations/`) :
   - programme-achats.pdf
   - programme-supply-chain.pdf

5. **Images de webinars** (à placer dans `public/`) :
   - webinar-altior.jpg

Les fichiers sont actuellement référencés dans le code mais n'existent pas encore.
